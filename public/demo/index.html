Embed Code

Add to Your Site in 2 Lines

No API key. No rate limits. Copy and paste into any HTML page. Works on React, Vue, static HTML, and WordPress.

<script src="https://api.b2bdataindex.com/widget.js"></script>
<div class="b2b-benchmark"
     data-market="usa"
     data-industry="technology">
</div>
Replace usa with any country slug and technology with any industry slug.
